## Overview

This is the personal portfolio website of **Mohamed Suhaib Nadakkavil (msn4u)**, a front-end focused developer and freelancer.  
The site showcases my skills, selected projects, services, and contact details, with a clean modern UI, responsive layout, and light/dark mode support.  
It is built purely with HTML and Tailwind CSS, optimized for use as a starting point for freelancing and profile optimization on platforms like Upwork.

## Features

- Fully responsive layout for mobile, tablet, and desktop
- Light / dark mode toggle with saved user preference
- Smooth scrolling, scroll animations, and back-to-top button
- Hero, About, Education, Skills, Projects, Services, Workflow, Testimonials, FAQ, and Contact sections
- Project gallery with category filters and modal-style project details
- Simple, clean Services section focused on HTML, CSS, Bootstrap, and Tailwind work
- SEO-friendly meta tags and favicon setup

## Tech Stack

- **HTML5**
- **Tailwind CSS**
- **Vanilla JavaScript** (animations, theme toggle, modals, filters)
- **vercel** (deployment)
